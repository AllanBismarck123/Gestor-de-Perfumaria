# Gestor de Perfumaria

### The application allows you to register cosmetic brands and transactions (purchase for stock replenishment or sales), in addition to allowing you to display all brands, all cosmetics, as well as their edition. The user can also monitor the profit for the last 12 months based on registered transactions.

### Business rules:

### A brand can only be deleted if there is no transaction registered with that particular brand.

### A cosmetic can only be registered if there is at least one brand registered to be selected.

### Usage tips are shown in dialog windows and there is also a screen just showing these rules and usage tips.
